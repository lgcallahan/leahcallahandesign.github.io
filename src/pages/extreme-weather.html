Extreme Weather

Website Design and Development / Junior Project Manager

Collaborative design project with the Digital Corps. Served as Junior Project Manager for the Architecture for Extreme Environments Website. Did not create any design work, but rather oversaw and led the project of eleven team members, and served as a liaison between team members and the Project Manager.

Duties as Junior Project Manager included maintaining project timeline, lead client meetings, gave feedback on design assets and other tasks, and frequently checked in with each of the members on tasks that needed to be completed and upcoming deadlines. Problem solved when project was behind deadline and created and planned a semester long timeline for website between five teams of video, development, design, user experience, and communication. Scheduled and organized student interviews.
